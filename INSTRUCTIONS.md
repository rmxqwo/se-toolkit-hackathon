## Вариант 2: Запуск с PostgreSQL (полная версия)

### Требования:

1. **Установите PostgreSQL** (если еще не установлен):
   - Скачайте с https://www.postgresql.org/download/windows/
   - Установите с паролем для пользователя `postgres`: `postgres`

2. **Создайте базу данных**:
   ```bash
   psql -U postgres
   CREATE DATABASE resume_builder;
   \q
   ```

3. **Проверьте файл .env**:
   Убедитесь, что в файле `.env` правильные настройки:
   ```
   DATABASE_URL=postgresql+asyncpg://postgres:postgres@localhost:5432/resume_builder
   ```

4. **Запустите приложение**:
   ```bash
   cd c:\Users\Eldar\Desktop\GitHub\software-engineering-toolkit\se-toolkit-hackathon
   python -m app.main
   ```

5. **Откройте браузер**:
   ```
   http://localhost:8000
   ```

## Как использовать

1. **Заполните личную информацию** (имя, email, и т.д.)
2. **Нажмите "💾 Save Profile"** - сохранит ваш профиль
3. **Добавьте навыки, опыт, проекты, образование** - используйте кнопки "+ Add"
4. **(Опционально) Вставьте описание вакансии** - для AI оптимизации
5. **Нажмите "✨ Generate Resume"** - создаст ваше резюме
6. **Скачайте PDF** - нажмите кнопку "📥 Download PDF"

## Возможные проблемы и решения

### Ошибка "User not found"
- Убедитесь, что вы сохранили профиль перед генерацией резюме

### Ошибка "Database connection failed"
- Проверьте, запущен ли PostgreSQL
- Проверьте настройки в файле `.env`

### Резюме не генерируется
- Откройте консоль браузера (F12) и посмотрите на ошибки
- Убедитесь, что заполнили хотя бы одно поле (опыт, навыки, проекты)

### PDF не скачивается
- Проверьте, что папка `generated_pdfs` существует
- Проверьте права доступа к папке
